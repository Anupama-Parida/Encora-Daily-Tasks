# Registration Form 
This task is a simple registration form built using **React**, **Vite**, and **manual form validation**.
This task validates user input in real-time and displays error messages dynamically.
## Features
- Validate **empty fields**
- Validate **email format**
- Validate **password length (min 6 characters)**
- Validate **confirm password matches**
- **Real-time error messages**
- **Disable submit button** until form is valid
- Responsive & clean UI
